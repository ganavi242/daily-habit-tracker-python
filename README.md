# 50 Days of Python 

Welcome to my 50 Days of Python Challenge  
This repository documents my journey of learning Python through **daily coding practice**, **basic programs**, **mini projects**, and **real-life problem-solving tasks**.

## 📌 About This Challenge
The goal of this challenge is to improve my Python programming skills by coding consistently for **50 days** and strengthening my understanding of:

- Print statements
- Input / Output
- Variables
- Conditional statements
- Loops
- Strings
- Lists
- Functions
- Basic problem solving
- Real-life mini projects

## 🎯 Objectives
- Build a strong Python foundation
- Practice coding daily
- Improve logic and problem-solving skills
- Create beginner-friendly mini projects
- Document my learning journey publicly on GitHub

## 📂 Challenge Structure
This challenge is divided into **two phases**:

### **Phase 1 – Basics Building (Day 1 to Day 30)**
- Focus on Python fundamentals
- **2 basic programs per day**
- Covers conditions, loops, strings, and simple logic

### **Phase 2 – Mini Projects & Real-Life Applications (Day 31 to Day 50)**
- Focus on applying Python concepts
- **1 mini project per day**
- Covers practical problem solving and beginner-friendly projects
